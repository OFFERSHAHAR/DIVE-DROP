🔄 איך ה-Workflow עובד?

הסבר מלא וקונקרטי של כל שלב בתהליך

🎯סקירה כללית

מה ה-Workflow עושה?
כשסוכן AI שלך צריך עזרה אנושית, הוא שולח הודעה ל-Slack. N8N קולט את ההודעה, מעבירה לטלגרם אלי, אני לוחץ כפתור, והודעה חוזרת לסוכן דרך Anthropic API.
🤖 Slack (AI Agent)
→
🔧 N8N
→
💬 Telegram (Me)
→
🧠 Anthropic API

📍תהליך שלב אחר שלב

1

🤖 הסוכן שלך שולח הודעה ב-Slack

כשיש בעיה שדורשת עזרה אנושית, הסוכן שלך שולח הודעה ל-Slack בפורמט JSON עם כל המידע החשוב:

{ "event": "human_handoff_requested", "session_id": "sesn_12345", "client_name": "גברת כהן", "topic": "בעיה בתשלום", "summary": "הלקוח לא יכול להשלים רכישה", "urgency": "high", "timestamp": "2025-01-15T14:32:00" }
⬇️
2

🔧 N8N Webhook מקבל את ההודעה

N8N כולל Webhook שמאזין לערוץ Slack מסוים. כשהודעה מגיעה, הוא מיד מעבד אותה.

מה קורה:
• N8N מקבל את JSON מ-Slack
• מדיווח שהודעה התקבלה
• מוכן למצב הבא
⬇️
3

📝 Parse JSON - שליפת המידע

N8N משתמש בקוד JavaScript כדי לשלוף את כל השדות מ-JSON:

const jsonData = JSON.parse(slackMessage); return { session_id: jsonData.session_id, client_name: jsonData.client_name, topic: jsonData.topic, summary: jsonData.summary, urgency: jsonData.urgency }
⬇️
4

💬 שליחת הודעה לטלגרם

N8N שולח הודעה אלי בטלגרם עם כל המידע + 2 כפתורים:

📞 בקשת Handoff חדשה

לקוח: גברת כהן
נושא: בעיה בתשלום
דחיפות: high

תקציר:
הלקוח לא יכול להשלים רכישה

כפתורים:
✅ הצטרף לשיחה
❌ דחה
⬇️
5

👆 אתה לוחץ כפתור בטלגרם

עכשיו זה בידיים שלך. אתה יכול:

שתי אפשרויות:

  • ✅ הצטרף לשיחה - אתה מסכים להצטרף לשיחה
  • ❌ דחה - אתה דוחה את הבקשה
⬇️
6

🔍 N8N בודק איזה כפתור לחצת

N8N מאזין לטלגרם וקולט את ה-callback מהלחיצה:

callback_data = "join_chat_sesn_12345" // או callback_data = "reject_sesn_12345"

אם בחרת "הצטרף" → המשך לשלב הבא
אם בחרת "דחה" → שלח הודעת "בקשה נדחתה"

⬇️
7

🧠 שליחה ל-Anthropic API

N8N שולח HTTP POST request ל-Anthropic Sessions API:

POST https://api.anthropic.com/v1/beta/conversations/sesn_12345/messages { "model": "claude-3-5-sonnet-20241022", "max_tokens": 1024, "messages": [{ "role": "user", "content": "🔔 מטפל אנושי הצטרף לשיחה" }] }
⬇️
8

💭 Claude מחזיר תשובה

Claude (ה-AI של Anthropic) מקבל את ההודעה ומחזיר תשובה:

{ "content": [{ "type": "text", "text": "שלום! התחברתי כעת ל-Slack וטלגרם. מטפל אנושי זה עתה הצטרף לשיחה כדי לעזור." }], "model": "claude-3-5-sonnet-20241022" }
⬇️
9

💬 N8N שולח תשובה לטלגרם

N8N לוקח את התשובה מ-Claude ושולח אותה אלי בטלגרם:

שלום! התחברתי כעת ל-Slack וטלגרם. מטפל אנושי זה עתה הצטרף לשיחה כדי לעזור.
⬇️
10

✅ סיום תהליך

N8N מעדכן את הכפתור בטלגרם שהתהליך הצליח:

✅ הצטרפת בהצלחה לשיחה!

✨ כל התהליך הסתיים! הסוכן יודע שמטפל אנושי הצטרף.

🔧כל הצמתים (Nodes)

שלב שם הצומת סוג מה זה עושה?
1 Slack Webhook Trigger Webhook מקבל הודעות מ-Slack
2 Parse JSON Code (JavaScript) שולף את כל השדות מ-JSON
3 Telegram Send Message Telegram שולח הודעה אלי בטלגרם
4 Telegram Callback Listener Telegram Polling מאזין לטלגרם ומחכה לחיצה
5 Extract Session from Callback Code (JavaScript) שולף את ה-session_id מהלחיצה
6 Is Join Action? IF Statement בדיקה: הצטרף או דחה?
7 Send Message to Anthropic HTTP Request שלח הודעה ל-Anthropic API
8 Send Response to Telegram Telegram שלח את תשובת Claude לטלגרם
9 Answer Callback Query HTTP Request עדכן את הכפתור בטלגרם

⭐נקודות חשובות

🎯 מה כל צומת עושה בדיוק?

  • Webhook: "שלום! יש הודעה בשביל אתה מ-Slack"
  • Parse JSON: "בואי נקח את כל המידע מהודעה"
  • Telegram Send: "אל, בואי אשלח לך הודעה מיוחדת"
  • Listener: "חכה... עכשיו אני מאזין למה אתה לוחץ"
  • Extract: "אתה לחצת על כפתור, בואי נראה איזה אחד"
  • IF Statement: "אם הצטרף → חזור להודעה, אם דחה → שלח דחייה"
  • Anthropic: "בואי נשלח הודעה ל-Claude על מה שקרה"
  • Response: "Claude עונה, בואי נחזיר את התשובה אליך"

14:31 - אני מקבל הודעה בטלגרם:

📞 בקשת Handoff חדשה

נושא: שאלה על הזמנה
תקציר: הלקוח שאל איפה ההזמנה שלו

[✅ הצטרף לשיחה] [❌ דחה]

"שלום! התחברתי כעת לשיחה. מטפל אנושי יוכל לעזור למשה בכל שאלה."

14:32:15 - אני מקבל בטלגרם:

שלום! התחברתי כעת לשיחה. מטפל אנושי יוכל לעזור למשה בכל שאלה.
