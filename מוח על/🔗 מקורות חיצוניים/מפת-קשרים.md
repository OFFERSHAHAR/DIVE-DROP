## 🔄 זרימות מידע

### זרימה 1: תמונות → ארכיון
```
צילום בשטח → D:\ תקיית תמונות → [ניב/עמוס/מיכאל] 
  ↓
כספת Obsidian → [[ניב 2024 מרץ]] (reference point)
  ↓
ניתן לאחזר לפרויקטים או דוקומנטציה
```

### זרימה 2: קורסים → ידע
```
חומרי קורס ב- D:\ → [[קלוד תיקית אובסידיאן]]
  ↓
קישור מ-_מפתח → [[לימודים-וקורסים]]
  ↓
עזרה ללמידה והוראה
```

```
קוד/פרויקט ב- D:\ → [[מוצרים -פרוייקטים - וסוכנים]]
```
