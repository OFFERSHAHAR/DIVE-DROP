✅ סטטוס ההגדרה

המצב: המערכת הוגדרה בהצלחה!

Windows Task Scheduler מוגדר להריץ את סוכן הידע כל שעתיים בתוך הרקע.

📦 מה נוצר

1️⃣ PowerShell Script

C:\Users\GamingPC\claude-knowledge-updater.ps1

זהו הscript שרץ כל שעתיים ומעדכן את תקיית הסוכן בObesidian.

2️⃣ Windows Task

Claude-Knowledge-Updater

משימה משוכללת שרץ אוטומטית כל שעתיים ללא הפסקה.

3️⃣ Log File

C:\Users\GamingPC\claude-knowledge-updater.log

קובץ רישום של כל הריצות וההודעות.

⚙️ איך זה עובד

Every 2 Hours: ┌──────────────────────────┐ │ Windows Task Scheduler │ └────────────┬─────────────┘ │ ▼ ┌──────────────────────────┐ │ PowerShell Script │ │ (claude-knowledge-...) │ └────────────┬─────────────┘ │ ▼ ┌──────────────────────────┐ │ Update Obsidian Vault │ │ (🤖 סוכן-Claude-Terminal) │ └────────────┬─────────────┘ │ ▼ ┌──────────────────────────┐ │ Write Log File │ │ (timestamp + status) │ └──────────────────────────┘

פעולות מדויקות:

  • בדוק אם תקיית הסוכן קיימת
  • עדכן timestamp בקובץ ה-index
  • בדוק וצריך לעדכן קבצי ידע
  • כתוב לקובץ log בהצלחה או שגיאה

👀 איך לראות את המשימה

דרך 1: Task Scheduler GUI

  1. פתח Task Scheduler (ממנו Start Menu)
  2. לחץ על Task Scheduler Library
  3. חפש Claude-Knowledge-Updater

דרך 2: PowerShell

Get-ScheduledTask | Where-Object {$_.TaskName -like "*Claude*"}

דרך 3: CMD

schtasks /query /tn "Claude-Knowledge-Updater"

📝 עקוב אחרי הלוגים

קובץ הלוג:

C:\Users\GamingPC\claude-knowledge-updater.log

קרא את הלוג בPowerShell:

Get-Content "C:\Users\GamingPC\claude-knowledge-updater.log" -Tail 20

או עם Real-time monitoring:

Get-Content "C:\Users\GamingPC\claude-knowledge-updater.log" -Wait

🔧 משימות נפוצות

➕ להפעיל את המשימה עכשיו

Start-ScheduledTask -TaskName "Claude-Knowledge-Updater"

⏸️ לעצור את המשימה

Stop-ScheduledTask -TaskName "Claude-Knowledge-Updater"

🔄 להפעיל מחדש

Restart-Computer

המשימה תרוץ אוטומטית בהפעלה הבאה.

🗑️ להסיר את המשימה (אם נדרש)

Unregister-ScheduledTask -TaskName "Claude-Knowledge-Updater" -Confirm:$false

🚀 המשך

עכשיו המערכת רצה!

המשימה תרוץ אוטומטית כל שעתיים בתוך הרקע. אתה יכול:

  • 👀 עקוב אחרי הלוגים לבדיקת עדכונים
  • 📝 ערוך את הscript כדי להוסיף יכולות
  • ⏰ שנה את התדירות (למשל לכל שעה)
  • 🔄 הפעל ידנית בכל זמן שרצית

⚙️ Advanced - שינוי התדירות

אם אתה רוצה לשנות את התדירות (למשל לכל שעה או כל 4 שעות):

כל שעה:

schtasks /change /tn "Claude-Knowledge-Updater" /ri 60

כל 4 שעות:

schtasks /change /tn "Claude-Knowledge-Updater" /ri 240

הערה: /ri הוא בדקות. 120 = 2 שעות, 60 = שעה, 240 = 4 שעות
