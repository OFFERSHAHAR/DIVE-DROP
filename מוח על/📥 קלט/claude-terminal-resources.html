⌨️פקודות Claude Terminal

התחלה בסיסית

פקודה תיאור
claude-code פתח Claude Code באופן אינטראקטיבי
claude-code "prompt here" הרץ prompt חד-פעמי
claude-code --version בדוק את הגרסה
claude-code --help ראה העזרה

עבודה עם קבצים

פקודה תיאור
claude-code -c file.js "task" ספק קבצים כקונטקסט
claude-code -c src/ "analyze" ספק תיקייה שלמה
cat file.js | claude-code pipe תוכן לClaude
claude-code > output.txt שמור פלט לקובץ

בחירת מודל

פקודה תיאור
claude-code --model claude-haiku-4.5 מודל מהיר וזול
claude-code --model claude-sonnet-4.6 מודל מאוזן
claude-code --model claude-opus-4.8 מודל החזק ביותר

Advanced

# הצג ספירת tokens claude-code --show-tokens "prompt" # שמור session claude-code --output-dir ./logs # הגדר timeout claude-code --timeout 300000
