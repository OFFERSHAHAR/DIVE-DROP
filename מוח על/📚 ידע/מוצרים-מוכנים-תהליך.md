# 🏭 קו ייצור המוצרים — תהליך ורישום

> השיטה: בונים מוצר → **בודקים שהוא באמת חי ועובד** → רק אז מחברים להאב → מנהלים משם.
> מחסן המוכנים: `D:\קלוד תיקית אובסידיאן\מוצרים מוכנים`

---

## 🔄 התהליך (4 שלבים)

```
1. מעבדה          2. בדיקת מוכנות        3. מחסן + האב         4. שליפה ללקוח
   בונים סוכן  →   חי? עובד מלא?     →    עובר ל"מוכנים"   →   מעתיקים,
   /מוצר          לא הוזה? לא          מתחבר להאב            מתאימים,
                  "אלוהים"?            מנוהל משם             מריצים
```

## 🚦 בדיקת מוכנות (Readiness Gate)
מוצר עובר הלאה **רק** כשכל אלה ✅:
1. **חי ורץ** — עולה בלי שגיאות, מגיב
2. **עובד מקצה לקצה** — התרחיש המרכזי עובד באמת
3. **לא הוזה** — מבוסס, לא ממציא
4. **לא "מרגיש אלוהים"** — נשאר בגבולות התפקיד, לא חורג להרשאות
5. **שגיאות ומקרי קצה** מטופלים
6. **סודות מאובטחים** — אין `.env` חשוף
7. **מתועד** — README: מה עושה / איך מריצים / מה מתאימים ללקוח

➡️ אחרי הכל ✅ → המוצר עובר ל[[מוצרים מוכנים]] → מתחבר ל[[project-agent-hub|האב]].
