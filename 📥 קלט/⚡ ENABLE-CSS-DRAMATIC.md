# ⚡ הפעל את CSS הדרמטי!

## 3 צעדים בלבד:

### 1️⃣ Obsidian Settings
- **⚙️ Settings → Appearance**
- Scroll down: **CSS Snippets**
- לחץ **Open snippets folder** (אם לא קיים)

### 2️⃣ בדוק CSS Snippet
- בדוק ש-`dramatic-theme.css` מופיע ברשימה
- לחץ **Enable** (כפתור עגול שחור)
- ✅ צריך להיות כחול/סגול

### 3️⃣ Reload
- `Ctrl+R` (Reload Obsidian)
- או סגור (`Alt+F4`) ופתח מחדש
