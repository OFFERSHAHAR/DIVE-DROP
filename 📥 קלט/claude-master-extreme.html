🔄 Advanced Workflows

Workflow #1: The Refactor Pipeline

TURN 1: "באל שיש לי קוד חזק שצריך refactor" └─ תן לי: 1. current pain points 2. success criteria 3. constraints TURN 2: "תן לי את ה-pattern בשורות 1-50" └─ אני: extract pattern ברוחניות TURN 3: "עכשיו apply בשורות 51-150" └─ אני: apply pattern + test

Workflow #2: The Debugging Pipeline

STEP 1: "שהתחל בstdout/log" └─ בדוק מה בעצם קורה STEP 2: "תן לי את הstrace/stack" └─ trace execution path STEP 3: "איזה hypothesis?" └─ אני אבדוק כל אחד STEP 4: "תקן את זה" └─ אני אעשה fix surgical

Workflow #3: The Feature Building Pipeline

PHASE 1: Architecture Review ├─ בקש: "design review לפני קודינג" ├─ תן: constraints + requirements └─ קבל: architecture diagram PHASE 2: Skeleton Build ├─ בקש: "build skeleton בדרגת strict" ├─ כולל: types + interfaces └─ ללא: implementation details PHASE 3: Impl + Tests ├─ בקש: "implement + tests" ├─ בconditions אחרונות └─ רץ: npm test PHASE 4: Docs + Edge Cases ├─ בקש: "documentation + edge cases" ├─ כולל: examples └─ רץ: final validation
