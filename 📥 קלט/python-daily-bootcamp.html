📅 יום 2: Variables - משתנים 10 דקות

🎯 המטרה

לשמור מידע בקופסאות (variables)

📖 הלימוד

# משתנה = קופסה שאתה שומר בה מידע name = "Alice" age = 25 height = 1.75 print(name) # Alice print(age) # 25 print(height) # 1.75

💡 מושגים

Assignment

x = 5 (שימת ערך בקופסה)

Types

string, int, float, bool

🎯 תרגיל

צור 3 משתנים: שם, גיל, עיר. תודפס את כולם:

name = "Your Name" age = 20 city = "Tel Aviv" print(name) print(age) print(city)
📅 יום 3: Data Types - סוגי מידע 10 דקות

📖 הלימוד

# STRING - טקסט name = "Python" print(type(name)) # # INTEGER - מספר שלם count = 42 print(type(count)) # # FLOAT - מספר עם נקודה price = 19.99 print(type(price)) # # BOOLEAN - נכון או לא נכון is_fun = True print(type(is_fun)) #
🎯 תרגיל

צור משתנה מכל סוג וודא את ה-type:

my_string = "Hello" my_int = 100 my_float = 3.14 my_bool = False print(type(my_string)) print(type(my_int))
📅 יום 4: Math Operations - חישובים 10 דקות

📖 הלימוד

# חיבור result = 5 + 3 # 8 # חיסור result = 10 - 4 # 6 # כפל result = 6 * 7 # 42 # חילוק result = 20 / 4 # 5.0 # חיסור עם שארית result = 17 % 5 # 2 # חזקה result = 2 ** 3 # 8 print(result)
🎯 תרגיל

חשב: (10 + 5) * 2 - 8

result = (10 + 5) * 2 - 8 print(result) # 22
📅 יום 5: String Operations - עבודה עם טקסט 10 דקות

📖 הלימוד

# Concatenation - הצמדת טקסט greeting = "Hello" + " " + "Python" print(greeting) # Hello Python # Repetition - חזרה laugh = "Ha" * 3 print(laugh) # HaHaHa # String length - אורך text = "Python" print(len(text)) # 6 # Slicing - חלק מהטקסט word = "Python" print(word[0]) # P print(word[0:3]) # Pyt
🎯 תרגיל

צור את המילה "PythonPython" בעזרת repetition:

result = "Python" * 2 print(result) # PythonPython
📅 יום 6: Input - קבלת מידע מהמשתמש 10 דקות

📖 הלימוד

# input() קורא משהו מהמשתמש name = input("What is your name? ") print("Hello, " + name) # המשתמש קורא: "Alice" # התוכנה תדפיס: Hello, Alice
🎯 תרגיל

שאל לגיל המשתמש והדפס "You are X years old":

age = input("How old are you? ") print("You are " + age + " years old")
