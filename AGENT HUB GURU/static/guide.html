🚀 פתח את הממשק ⚡ Quick Start
⚡ התחלה מהירה 🤖 הסוכנים 🖥️ שימוש בממשק 🔌 API 💡 טיפים 🔧 פתרון בעיות

התחלה מהירה — 3 שלבים
1
קבל מפתח API מ-Anthropic
עבור לדף console.anthropic.com → API Keys → Create Key.
המפתח נראה כך: sk-ant-api03-...
💡 צריך חשבון Anthropic עם אמצעי תשלום פעיל. עלות ממוצעת: ~$0.01–0.05 לשיחה.
2
הגדר את המפתח ב-PowerShell
פתח PowerShell בתיקיית הפרויקט והרץ:
PowerShell 
$env:ANTHROPIC_API_KEY = "sk-ant-...כאן המפתח שלך..."
⚠️ אל תשמור את המפתח בתוך קובץ קוד או Git — השתמש במשתנה סביבה בלבד.
3
הפעל את השרת ופתח בדפדפן
PowerShell 
cd C:\Users\GamingPC\smart-agents
python server.py
✅ פתח בדפדפן: http://localhost:5000
🤖
שלושת הסוכנים
📋
סוכן #1 — מסווג לידים
מקבל טקסט חופשי (מטופס, אימייל, שיחה) ומנתח אותו לסיווג מובנה בפורמט JSON.
🔴 URGENT / WARM / COLD ⭐ HIGH / MED / LOW 📞 פעולה מומלצת
🎧
סוכן #2 — שירות לקוחות
צ'אט רב-שיחה עם זיכרון, מסביר פתרונות, מתאם פגישות ומטפל בתמיכה ראשונית.
💬 Multi-turn chat 🏭 ניסיון תעשייתי 📅 תיאום פגישות
💼
סוכן #3 — מכירות (אלון)
שיחת מכירה מלאה עם שיטת SPIN, טיפול בהתנגדויות וסיכום הצעת ערך.
🎯 SPIN Selling 💰 ROI חישוב 📋 סיכום הצעה
🖥️
מדריך שימוש בממשק
1
טאב "מסווג לידים"
שלב א': לחץ על אחד מכפתורי הדוגמה (🔴 ליד דחוף / 🟡 ליד חמים / 🟢 ליד קר) לטעינת דוגמה מוכנה.

שלב ב': אפשר לערוך את הטקסט בחלון הכתיבה — הכנס טקסט אמיתי מהלידים שלך.

שלב ג': לחץ "🔍 סווג ליד" וחכה כ-3-5 שניות.

תוצאה: תקבל כרטיסיות עם דחיפות, תחום פתרון, פוטנציאל עסקי, פעולה מומלצת ועוד.
2
טאב "שירות לקוחות"
שלב א': לחץ "💬 התחל שיחה חדשה" — הסוכן יפתח עם הודעת פתיחה אוטומטית.

שלב ב': הקלד שאלה בתחתית הצ'אט ולחץ שלח (או Enter).

טיפ: לחץ על אחד מ"רעיונות לשאלות" בסרגל הצד לקבלת השראה.

שלב ג': לסיום — לחץ "🔚 סיים שיחה".
3
טאב "סוכן מכירות"
שלב א': מלא פרטי הליד בסרגל הצד:
  • שם הלקוח (לדוגמה: דני לוי)
  • שם החברה (לדוגמה: תעשיות אלפא)
  • כאב עיקרי (לדוגמה: ERP ישן, תהליכים ידניים)

שלב ב': לחץ "📞 התחל שיחת מכירה" — "אלון" יפתח עם פתיח מותאם ללקוח.

שלב ג': שחק את תפקיד הלקוח — נסה התנגדויות מרשימת "משחקי תפקידים".

שלב ד': לחץ "📋 סיכום הצעת ערך" לקבל סיכום פנימי מהשיחה.
🔌
API Reference — לחיבור מערכות חיצוניות

כל ה-endpoints מקבלים ומחזירים JSON. הבסיס: http://localhost:5000

Method Endpoint תיאור Body (JSON)
POST /api/classify סיווג ליד { lead_text: "..." }
POST /api/cs/session פתיחת שיחת שירות לקוחות {}
POST /api/cs/chat שליחת הודעה בשיחה { session_id, message }
DELETE /api/cs/session/:id סגירת שיחת שירות לקוחות —
POST /api/sales/session פתיחת שיחת מכירה { name, company, pain_point }
POST /api/sales/chat שליחת הודעה בשיחת מכירה { session_id, message }
POST /api/sales/summary יצירת סיכום הצעת ערך { session_id }
DELETE /api/sales/session/:id סגירת שיחת מכירה —
דוגמת שימוש — curl 
curl -X POST http://localhost:5000/api/classify \
  -H "Content-Type: application/json" \
  -d '{"lead_text": "שם: דני | צריך מערכת CRM בדחיפות"}'

טיפים לשימוש יעיל
📝
סיווג לידים — כמה שיותר מידע, כל כך יותר טוב
הכנס לסוכן את שם הלקוח, שם החברה, מספר עובדים, מה שאמר בדיוק ואיך הגיע. ככל שיש יותר הקשר — הסיווג מדויק יותר.
🎭
מכירות — שחק התנגדויות אמיתיות
הסוכן מאומן על ההתנגדויות הנפוצות ביותר. נסה: "זה יקר לנו", "כבר יש לנו ספק", "אנחנו לא מוכנים לשינוי" — ותראה איך הוא מגיב.
🔗
חיבור בין סוכנים — תהליך מכירה מלא
הזרימה המומלצת: סווג ליד ← שירות לקוחות ← מכירות.
קח את פרטי הליד מסוכן #1 והכנס אותם לסוכן #3 לקבלת פתיח מותאם.
💾
שמירת מפתח API באופן קבוע
כדי שלא תצטרך להכניס את המפתח בכל פעם, הוסף לקובץ $PROFILE של PowerShell:
$env:ANTHROPIC_API_KEY = "sk-ant-..."
📊
יומן שיחות — בנה בסיס ידע
שמור את תוצאות הסיווג וסיכומי המכירות בגיליון. לאחר 20-30 שיחות תוכל לראות דפוסים — אילו לידים מסתגרים, אילו התנגדויות נפוצות וכו'.
🔧
פתרון בעיות נפוצות
!
שגיאה: "ANTHROPIC_API_KEY לא מוגדר"
הפתרון: הכנס את המפתח לפני הפעלת השרת: 
$env:ANTHROPIC_API_KEY = "sk-ant-..."
python server.py
חשוב: יש לעשות זאת באותו חלון PowerShell שממנו מפעילים את השרת.
!
